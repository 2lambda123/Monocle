Typeclasses

Monocle defines typeclasses to fetch generic optics implicitly. You can find more information about typeclass pattern in cats.

Here is the list of typeclasses defined in Monocle:

  • At
  • Cons
  • Cons1
  • Curry
  • Each
  • Empty
  • Field1 to Field6
  • Index
  • Plated
  • Possible
  • Reverse
  • Snoc
  • Snoc1
